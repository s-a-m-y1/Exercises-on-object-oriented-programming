## ğŸ” Cryptographic Engine (5-Layer Security)
To protect sensitive financial data, the system utilizes a custom-built **multi-stage encryption pipeline**:
1. **XOR Cipher:** Fundamental stream encryption using a private key.
2. **Noise Injection:** Strategic random padding to obscure data patterns.
3. **Bitwise Casing:** Inversion logic to alter character byte representation.
4. **Nested XOR:** A secondary encryption pass for increased complexity.
5. **Hexadecimal Encoding:** Final storage in hex format to render raw data files unreadable to standard text editors.
